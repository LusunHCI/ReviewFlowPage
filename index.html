ReviewFlow is a system that provides process-oriented and contextual scaffolding using LLM to support the novices peer review workflow. It provides an AI-driven workflow that scaffolds novices with contextual reflections to critique and annotate submissions, in-situ knowledge support to assess novelty, and notes-to-outline synthesis to help align peer reviews with community expectations.
